# Roteiros Madeira — Protótipo

Este repositório contém um protótipo estático (HTML/CSS) de um marketplace de roteiros e experiências para a Madeira e Porto Santo.

## Como usar
- Abra `index.html` num browser moderno.
- Navegue pelas páginas: `experiencias.html`, `places-*.html`, `detail.html`, `cart.html`, `favorites.html`, `itinerary.html`.
